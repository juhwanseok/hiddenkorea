# 참가 신청서 초안 (접수 페이지 입력용)

> 접수: api.visitkorea.or.kr → 회원가입 → "2026 관광데이터 활용 공모전(웹·앱 구현 부문)" 선택. **7/17까지 제출 목표.**
> 실제 입력 항목은 접수 폼 확인 후 조정. 팀원 정보는 접수 시 생략 가능(1차 심사 전까지 전원 가입 필요).

**서비스명**: 숨은한국 (HiddenKorea)

**선택 지정과제**: 2번 — 유명 관광지 중심의 관광객 쏠림 현상으로 인한 특정 명소의 오버투어리즘(과밀화) 문제

**서비스 한 줄 소개**:
방문 예정일의 관광지 혼잡도를 AI로 예측하고, 취향은 유지한 채 혼잡만 제거한 '비슷하지만 한적한' 대안 관광지와 여행 코스를 추천 근거와 함께 제공하는 오버투어리즘 분산 웹 서비스

**기획 요약 (500자 내)**:
유명 명소 쏠림은 여행자 경험 저하, 지역 주민 피해, 인근 저평가 관광지의 기회 상실이라는 3중 손실을 만듭니다. 숨은한국은 ① 한국관광공사 관광지 집중률 예측 데이터와 자체 머신러닝 모델을 결합한 하이브리드 혼잡도 엔진으로 전국 관광지의 날짜별 혼잡도를 예보하고, ② TourAPI 관광지 정보의 AI 임베딩 유사도 분석으로 "매력은 비슷하지만 한적한" 대안 관광지를 유사도·예상 혼잡·거리 근거와 함께 추천하며, ③ 대안을 반영한 하루 코스를 자동 재구성합니다. 한국관광공사 OpenAPI(국문 관광정보, 관광지 집중률 예측, 지역별 방문자수, 연관 관광지)가 서비스의 핵심 경로에 필수적으로 활용되며, 기상청·서울시 공공데이터로 예측을 보강·검증합니다. 관광 수요를 지우지 않고 옮김으로써 여행자 만족과 지역 균형 발전을 동시에 달성하는 것이 목표입니다.

**활용 데이터(공사 OpenAPI)**: 국문 관광정보 서비스, 관광지 집중률 방문자 추이 예측 정보, 빅데이터 지역별 방문자수, 관광지별 연관 관광지 정보
**외부 데이터**: 기상청 단기예보, 서울 실시간 도시데이터(검증용), 공휴일 정보

**해시태그**: #혼잡도 분석 #과밀지역 우회 #숨은 명소 #코스 추천

**서비스 형태**: 반응형 웹 (모바일 우선)
