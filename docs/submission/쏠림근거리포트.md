# 오버투어리즘 쏠림 — 정량 근거 리포트

> 한국관광공사 ‘관광지 집중률 방문자 추이 예측’ 데이터 **223,350건**(전국 관광지 **7,445곳** × 30일)을 분석. ‘쏠림은 실재하지만, 옮기면 풀린다’를 재현 가능한 숫자로 증명한다. 재현: `ml/eval/analyze_concentration.py`

## 한 줄 요약 (신청서·발표 첫 문장용)

> **같은 날, 같은 동네에서도 절반이 넘는 경우(56.7%)에 ‘매우혼잡’ 명소와 ‘한산한’ 명소가 동시에 존재한다. 인기 명소조차 날짜만 바꾸면 혼잡이 평균 56.8% 낮아진다.** 쏠림은 수요 부족이 아니라 **정보 부족에 의한 분배 실패** — 그래서 예측·대안 추천으로 옮길 수 있다.

## 핵심 지표

| 지표 | 값 | 의미 |
|---|---|---|
| ‘혼잡’ 이상(60↑) 관광지-날짜 비율 | **27.5%** | 쏠림은 실재 — 4곳 중 1곳 이상이 붐빔 |
| ‘매우혼잡’(80↑) 비율 | 11.3% | 극심한 과밀도 상시 발생 |
| ‘여유·보통’(40↓) 관광지-날짜 비율 | **47.9%** | 수요를 받아줄 ‘숨은’ 공급이 절반 |
| 같은 날·같은 시군구, 매우혼잡·한산 **공존** 비율 | **56.7%** | 옮길 곳이 바로 옆에 있음(공간 이동) |
| 인기 명소의 최다일→최저일 혼잡 감소 | **56.8%** (95→41) | 날짜만 바꿔도 붐빔 회피(시간 이동) |

## 이 숫자가 말하는 것 — 우리 서비스의 두 레버와 정확히 일치

1. **쏠림은 실재한다** — 전국 관광지-날짜의 27.5%가 ‘혼잡’ 이상, 11.3%가 ‘매우혼잡’.

2. **대안은 이미 있다** — 47.9%가 ‘여유·보통’. 비어 있는 ‘숨은’ 관광지가 절반.

3. **공간 이동 여력** — 같은 날·같은 시군구에서 붐빔과 한산이 **56.7%** 확률로 공존. 멀리 갈 필요 없이 **바로 옆 대안**으로 분산 가능 → 서비스의 ‘대안 매칭’이 직접 겨냥.

4. **시간 이동 여력** — 인기 명소(1,420곳)도 가장 한산한 날은 혼잡이 평균 **56.8% 낮다**(95→41). → 서비스의 ‘30일 혼잡 예측·달력’이 직접 겨냥.

> 결론: 문제(쏠림)와 해법(공간 대안 + 시간 예측)이 **같은 데이터 위에서 성립**한다. 여행자는 이 정보를 몰라서 못 옮길 뿐 — 그 격차를 우리가 메운다.

## 한계 (정직 고지)

- 집중률은 공사의 예측 스냅샷(향후 30일)이며 정규화 비율(0~100) → 방문자 절대수가 아닌 혼잡도 기준.

- 계절 전체가 아닌 현 시점 30일 리듬 → 장기 계절성은 별도 검증 과제.
