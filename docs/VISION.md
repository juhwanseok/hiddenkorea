# VISION — 숨은한국

**미션**: 여행의 만족과 지역의 균형을 동시에 — 수요를 지우지 않고 옮긴다.

**3년 그림**: 개인 여행자의 분산 의사결정 도구(B2C)에서 출발해, 분산 효과를 측정·리포팅하는 지자체 관광정책 인프라(B2G)로 확장한다. 한국관광공사 데이터 생태계 위에서 "혼잡"이라는 단일 축을 가장 깊게 파는 서비스가 된다.

**가치 순서**: 여행자 경험 > 지역 상생 > 기술 과시. AI는 언제나 수단이다.

**우리가 아닌 것**: 종합 여행 슈퍼앱 ✗ / 리뷰 플랫폼 ✗ / 실시간 관제 시스템 ✗
